Seafood Shop

Seafood Shop is a static web project that showcases a variety of seafood products through a clean, interactive gallery. Built with HTML, CSS, and JavaScript, this project demonstrates responsive design, asset organization, and simple frontend interactivity suitable for learning and portfolio purposes.
